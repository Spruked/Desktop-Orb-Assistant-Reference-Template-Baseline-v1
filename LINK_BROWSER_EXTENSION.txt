To use the LLM Governance Layer browser extension:

1. Open Chrome or Edge and go to chrome://extensions or edge://extensions.
2. Enable Developer Mode (toggle in the top right).
3. Click "Load unpacked" and select the folder:
   new orb/governance_layer/extension
4. The extension will now be active on supported LLM sites (OpenAI, Claude).

You can edit extension/governance.css, content.js, or popup.html for customization.
